Strengths and weaknesses of your final project

Strengths

  • The layout of the site is easy to follow and looks clean.

  • The background, font size and colour makes the site more readable.

Weaknesses

  • Having more time for the project would have been beneficial

  • The use of JavaScript would have been better to make the website more interactive

  • Not enough knowledge using other and better software such as Saas and Flexbox to help
    improve the look of the website.

Further changes to the site would be...

  • To add some animation to the home page

  • Using a trendier font would make the site look more professional

  • Adding a picture on the About page would be nicer to relate more to the user.

  • The use of illustrations on the navigation bar would have be more professional and make
    the site more accessible to those who have difficulties reading.
